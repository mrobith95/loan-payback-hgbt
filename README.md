# loan-payback-hgbt
Basic ML-based loan payback prediction. Originally a kaggle notebook for kaggle playground competition. Medium article for detailed explanation also available [here](https://medium.com/@mrobith95/loan-payback-prediction-using-histogram-gradient-boosting-trees-d93afa7fc961).

This jupyter notebook mainly focuses on in-depth modelling including feature selection and hyperparameter tuning, while also perform basic analysis routine such as data wrangling and ordinal encoding.
